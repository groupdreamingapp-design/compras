# Gestión de Compra (Propuesta)

Plataforma de gestión de compras, estrategia y auditoría para retail/gastronomía.

## Características

- **Dashboard**: KPIs en tiempo real (Inflación, Ahorro, Cumplimiento).
- **Auditoría**: Watchlist de precios, Impacto en CMV y Matriz de Proveedores.
- **Estrategia**: Análisis ABC (Pareto) y evolución de precios.
- **Rentabilidad**: Ingeniería de Menú (Matriz BCG).
- **Importar/Exportar**: Gestión masiva de insumos y proveedores vía CSV.

## Stack Tecnológico

- Next.js 16 (Turbopack)
- React
- Tailwind CSS
- SQLite (sql.js)
- Firebase (Auth & Firestore)
- Recharts
